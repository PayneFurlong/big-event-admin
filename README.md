# Big Event Admin

基于 Vue3 构建的后台内容管理系统（CMS）。

## 项目简介

该项目为一个后台管理系统，实现用户登录、文章分类管理、文章发布与管理等功能。

主要用于练习 Vue3 组件化开发及前端工程化实践。

## 技术栈

- Vue3
- Vite
- Pinia
- Vue Router
- Axios
- Element Plus

## 项目功能

- 用户登录认证
- 文章分类管理
- 文章发布与管理
- 富文本编辑
- 封面图片上传与预览
- Token 鉴权与路由守卫
